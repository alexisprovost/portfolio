# Portfolio [![Vercel](https://img.shields.io/github/deployments/alexisprovost/portfolio/production?label=vercel&logo=vercel)](https://vercel.com/m19cas-projects/alexisprovost.com)

Personal portfolio of Alexis Provost

## Commands

```bash
bun install   # Install dependencies
bun dev       # Start dev server (port 5173)
bun run build # Build for production
```

## Stack

React 19 + TypeScript + Vite + Tailwind CSS v4 + Framer Motion
